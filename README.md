# 庚元曆 (Gyeongwon-ryeok)

易은 曆이다.

## 개요
庚元曆은 동양의 전통적인 역학 체계(60갑자, 64괘)를 바탕으로 현대의 태양력과 조화를 이루도록 설계된 만세력 애플리케이션입니다. 360일을 한 주기로 하는 독특한 6주(六周) 시스템을 시각화하여 제공합니다.

## 주요 기능
- **엑셀 형태의 만세력 그리드**: 날짜별 간지(干支), 괘(卦), 요일 등을 한눈에 파악할 수 있는 격자형 UI.
- **6주 시스템 (Ju System)**: 복주(復周), 임주(臨周), 태주(泰周) 등 주별 탭을 통한 시간 흐름 추적.
- **날짜 컨트롤**: 연/월/일 단위의 이동 기능을 통해 과거 및 미래의 역법 정보 조회.
- **역법 메타데이터**: 역원(曆元), 세수(歲首), 기준점(선천/후천) 등 상세 역학 정보 제공.

## 기술 스택
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Fonts**: Noto Serif KR, Noto Sans KR (Google Fonts)
- **Data**: JavaScript 객체를 이용한 64괘 및 24절기 데이터 관리

## 설치 및 실행
특별한 빌드 과정 없이 웹 브라우저에서 `index.html` 파일을 열어 바로 실행할 수 있습니다.

```bash
# 로컬에서 실행 예시 (Python 사용 시)
python -m http.server 8000
```
이후 `http://localhost:8000`으로 접속하십시오.

## 파일 구조
- `index.html`: 메인 어플리케이션 구조 및 UI 레이아웃
- `app.js`: 역법 계산 로직, 데이터 정의 및 DOM 렌더링
- `style.css`: 전체 디자인 및 그리드 레이아웃 스타일
- `selected_style.css`: 선택된 항목이나 특정 상태를 위한 추가 스타일
